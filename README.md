# 🏋️ FitCoach (MVP)

FitCoach는  
**AI 도구들을 활용해 기획 → 디자인 → 구현까지 진행하는  
헬스 모바일 애플리케이션 MVP 프로젝트**이다.

이 프로젝트의 목적은  
완성된 서비스보다 **AI를 개발 보조 도구로 어떻게 활용할 수 있는지**를 실험하는 데 있다.

## 🤖 AI 도구 활용 방식

이 프로젝트는 개발 전 과정을 다음과 같이 나누어 AI를 활용했다.

- 📝 **기획 · 설계**
  - ChatGPT를 활용해
    - 서비스 컨셉 정리
    - MVP 범위 정의
    - 사용자 흐름 및 AI 코치 설계 진행

- 🎨 **디자인 · UX**
  - Google Stitch를 활용해
    - 와이어프레임 생성
    - 화면 레이아웃 및 UX 구조 구상

- 💻 **개발**
  - AI Codex를 활용해
    - 컴포넌트 구조 설계 보조
    - 반복 코드 작성
    - 로직 구현 보조

AI는 판단을 대신하지 않고,  
**개발자의 결정을 빠르게 돕는 도구로 사용**했다.

## 🧠 서비스 개요

FitCoach는  
**운동 기록 + AI 추천 + 명확한 이유 설명**을 중심으로 한  
헬스 모바일 애플리케이션이다.

- 사용자는 직접 운동을 기록한다
- AI는 기록을 해석해 다음 운동을 추천한다
- 모든 추천에는 이유가 포함된다

## 📦 MVP 범위

- 온보딩 / 프로필
- 운동 루틴 및 세션 기록
- 홈 대시보드 요약
- 히스토리 / 세션 상세
- 텍스트 기반 AI 코치

(커뮤니티, 식단, 웨어러블 연동은 MVP 범위에서 제외)
